«TEKLIPYAZ» is a mobile application that provides detailed information about institutions and feedbacks.

How to use it?

In order to use the application you need:

1. sign up

2. choose an institution

3. write a review and put a few stars, if the institution liked it, or not put it.

After the request is processed, a dialog box appears with the application logo and the words "Thank you for your feedback!".

More Information

The application was officially released on November 11, 2017 in the Play Market. Developed by developers of Teklipyaz Group in 2017.

The application is available in Turkmen, Russian and English languages.
